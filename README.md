# iGEM Wiki Redesign

This project is a redesign of the iGEM Wiki homepage, built using HTML, CSS, and JavaScript.

---

## Project Structure
- `html/` → All HTML files
- `css/` → All CSS files
- `js/` → All JavaScript files
- `images/` → Images used in the homepage
- `design_canva/` → Original Canva design files used as reference
